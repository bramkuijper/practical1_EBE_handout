2.2 A brief introduction to programming in R

While you will be learning a lot more about R in the associated statistics module taught by Dr Erik Postma, we will be focusing less on the statistical analysis. Rather, we will use some programming basics provided by R to implement our own model of an evolutionary process.

To this end, we need to know something about the ingredients that simple programs use:

2.2.1 Variables

A variable provides us with a named storage of data. A variable in R can contain any type of data (e.g., a list of values, a single number, a string of text, etc). We assign data to a variable using a = or <- operator. For example:

# create a variable named var.1 and assign a value to it using the <- operator
var.1 <- 25000

# create a variable named var.2 and assignment a vector, c(), of numbers to it using the = operator
var.2 = c(0,1,9.5,200)

You can also easily reassign new values to variables:

# print the value of a variable, showing its current value
print(var.1)
## [1] 25000
# reassign a new value to this variable, the previous value is now gone
var.1 <- 900

# print the new value
print(var.1)
## [1] 900

You can always check the type of your variable using the typeof() function:

typeof(var.1)
## [1] "double"
typeof(var.2)
## [1] "double"
